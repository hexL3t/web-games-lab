# Web Games Lab

A collection of educational games built using HTML, CSS, and JavaScript.

This repository is designed to support:
- JavaScript programming lessons
- Web Design & Development curriculum
- Game development fundamentals
- Interactive coding activities
- Beginner to intermediate coding projects

Each project is structured so it can be:
- explored as a complete game
- broken down into teachable lessons
- extended with new features and challenges

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## Project Goals

This repository aims to:
- teach programming through interactive projects
- provide reusable classroom examples
- demonstrate real-world web development concepts
- encourage creativity and experimentation

---

## Educational Focus

Projects may include:
- starter templates
- completed examples
- challenge extensions
- lesson breakdowns
- teacher notes
- student activities
